# MENARA.io

Climb the tower to escape the rising lava! Be the last one standing.

https://ongyean.itch.io/menara-io

### 🎮 Controls

| Action | PLAYER 1 (Blue) | PLAYER 2 (Orange) |
| :--- | :--- | :--- |
| **Move** | **A / D** | **J / L** |
| **Jump** | **W** | **I** |
| **Plunge** | **S** | **K** |

---

### ✨ POWERUPS
*   **FLY**: Gravity-defying flight! Hold Jump to soar. Only active for 5 seconds.
*   **DOUBLE JUMP**: Leap again while in mid-air.
*   **FLASHBANG**: Blinds your opponent's screen.

---

### 💡 GAME MECHANICS
*   **Combat**: Plunge onto opponents from above to stun them.
*   **Grace Period**: Stuns and Flashbangs are disabled until the lava starts rising—get into position!
*   **Trolls**: Watch out for random events like "Raining Cats and Dogs" or "What The Flip!"

## 🛠️ Technical Info
Built with **Godot 4.x**.
- **Character Model**: Scotty (scaled and squashed for maximum aesthetic).
- **Environment**: Procedurally generated infinite tower and platforms.
- **VFX**: Custom shaders for lava, vignettes, and screen-space effects.
